# QCA Learning Lab

## Descripción

Este repositorio documenta el proceso de aprendizaje sobre Qualitative Comparative Analysis (QCA), análisis de conjuntos, R y metodologías configuracionales aplicadas al estudio de políticas públicas, organizaciones y sistemas complejos.

El objetivo no consiste únicamente en aprender una técnica estadística, sino en construir un laboratorio de investigación reproducible donde se integren teoría, datos, código y reflexión metodológica.

---

## Objetivos

- Aprender QCA utilizando R.
- Documentar cada ejercicio paso a paso.
- Construir una biblioteca comentada de artículos.
- Reproducir investigaciones publicadas.
- Desarrollar habilidades en Git y GitHub.
- Explorar el diálogo entre QCA, la implementación de políticas públicas y los sistemas complejos (MCS).
